# Changelog

## 0.0.1

- Initial release
- Inline agent for writing code with `CMD+I` / `Ctrl+I`
- Support for Copilot, Anthropic, OpenAI, Google, Mistral, Groq, xAI, OpenRouter, and Together providers
- Project context via `CLAUDE.md` and `AGENT.md` files
- Inline diff review with accept (`CMD+Enter`) and reject (`CMD+Backspace`)
